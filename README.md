# TDEE 代谢修复工具站 MVP

基于 EWMA 算法的自适应 TDEE 计算器，专为代谢适应修复和 GLP-1 药物停用后恢复设计。

## 功能特点

- 🧮 **动态自适应 TDEE 计算** - 基于真实数据反向推导，而非静态公式
- 📊 **代谢缺口量化** - 直观展示理论与实际 TDEE 的差距
- 🔄 **反向饮食导航** - 智能周度热量调整建议
- 💊 **GLP-1 模式** - 专为停药用户优化的高蛋白质推荐
- 🔒 **隐私优先** - 所有数据存储在本地浏览器

## 核心算法

### EWMA 平滑
```
Trend_t = α × W_t + (1 - α) × Trend_{t-1}
```
α = 0.1 用于过滤每日水分波动

### TDEE 反推
```
TDEE = 摄入热量 - (体重变化 × 7700 kcal/kg)
```

### 反向饮食决策
- 体重变化 < 0.2% → 增加热量
- 0.2% ≤ 变化 ≤ 0.5% → 保持
- 变化 > 0.5% → 减少热量
